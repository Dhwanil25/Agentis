# Agentis

Agentis is an open-source AI agent that breaks any task into a pipeline of
composable skills — Planner, Architect, Coder, Reviewer, Tester, Documenter,
Writer, Analyst, and more — and runs them in sequence with real-time streaming
output at every step.

Instead of sending one giant prompt and waiting for a response, Agentis
orchestrates each skill as a focused expert: the Planner breaks down the task,
the Architect designs the structure, the Coder writes the implementation, the
Reviewer catches bugs, and the Documenter explains it all. Each skill sees the
previous steps' output as context, so the chain builds on itself.

Inspired by obra/superpowers (85k+ stars on GitHub). Powered by Claude API.
